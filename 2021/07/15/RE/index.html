汇编基础

寄存器

是CPU内部存储值的地方，下图是x86-64架构CPU的寄存器。

所有寄存器图

常用64位寄存器图

其中有几个固定使用的寄存器：

  1. RBP:栈底寄存器
  2. RSP:栈顶寄存器
  3. RIP:指向当前运行到指令的地址

内存

内存是用于程序运行时存储数据的地方。操作系统会自动为每个应用程序分配一定大小的虚拟内存空间，让程序认为自己在一块独立的内存空间运行。

常见汇编指令

在汇编中，用[]表示操作内存。

汇编指令示例 实际作用 等价代码
mov rax,rbx 用于赋值 rax=rbx
add/sub rax,rbx 用于加/减法 rax+=rbx/rax-=rbx
and/xor/or rax,rbx 用于与/异或/或 rax&=rbx/rax^=rbx/rax|=rbx
push rax 压栈 rsp+=8;*rsp=rax
pop rax 出栈 rax=*rsp;rsp-=8
call rax 调用函数 push rip;jmp rax;
ret 从函数返回 pop rip;
cmp rax,rbx 比较两个数 rax-rbx,不保留结果,只修改flags寄存器
test rax,rbx 比较两个数 rax&rbx,不保留结果,只修改flags寄存器

各种跳转指令

汇编指令示例 英文 实际作用
jmp jump 跳转
jz jump if zero 为0时跳转
jnz jump if not zero 不为0时跳转
jg jump if greater 有符号数大于跳转
jl jump if less 有符号数小于跳转

函数调用约定

64位/windows默认

函数参数依次放入寄存器RCX,RDX,R8,R9,栈

返回值放入RAX

64位/linux

函数参数依次放入寄存器RDI,RSI,RDX,R8,R9,…

返回值放入RAX

32位

函数参数压入栈中，返回值放入eax。

一般由函数调用者或函数内部平衡栈。

花指令

花指令是指在程序中加入一些无用的汇编指令干扰反编译器的工作。一般需要patch掉花指令后再反编译。
